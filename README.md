#  El Problema de los Tres Cuerpos: Coreografias de N-Cuerpos

Este proyecto explora una solución especial del problema de los tres cuerpos en la cual tres masas iguales siguen una trayectoria periódica en forma de ocho, conocida como una **coreografía celeste**.

##  Objetivo

Analizar la **estabilidad** de esta órbita frente a perturbaciones como:
- Pequeñas variaciones en las masas  
- Influencia gravitacional de un cuarto cuerpo externo  

## Pregunta de investigación

- ¿Qué tan estable es la coreografía en forma de ocho bajo perturbaciones?  
- ¿Cuándo el sistema pasa de un comportamiento periódico a uno caótico?

## Métodos

Se resuelven numéricamente las ecuaciones de movimiento de Newton mediante:
-
- Integración Leapfrog  

## Experimentos

- Órbita ideal en forma de ocho  
- Perturbaciones en las masas  
- Influencia de un cuerpo externo  

## Resultados

Incluyen:
- Gráficas de trayectorias  
- Animaciones  
- Análisis de estabilidad orbital
