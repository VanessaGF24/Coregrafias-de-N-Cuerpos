# Coregrafias-de-N-Cuerpos
Este proyecto explora una solución especial del problema de los tres cuerpos, en la cual tres masas iguales describen una trayectoria periódica en forma de ocho, conocida como coreografía celeste. El objetivo es analizar la estabilidad de esta órbita frente a perturbaciones, como pequeñas variaciones en las masas, entre otras.
